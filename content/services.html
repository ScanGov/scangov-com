Overview

ScanGov Services provides expert consulting and implementation support to help government organizations maximize their digital experience investments. Our services complement the ScanGov platform by providing hands-on expertise, strategic guidance, and accelerated implementation.

Accessibility Compliance Acceleration

Passing automated audits is a great start to full WCAG compliance. Once you achieve perfect scores on the automated reviews we will run the additional manual audits to get to complete coverage.

  • WCAG 2.1 AA manual review to insure complete compliance
    • Screen reader testing using common tools on real hardware
    • Color contrast and visual accessibility analysis
    • Mouseless navigation review
    • Focus highlighting analysis
    • Zoom, font size customization review
  • Accessibility training for internal teams
    • Tips on how to use the most popular assistive tech so your team can periodically run your own manual accessibility reviews
  • Duration: 3-6 weeks | Investment: $18,000-$40,000

Performance Optimization Sprint

  • Core Web Vitals improvement (LCP, FID, CLS)
  • Report on how your properties compare to peer organizations
  • Site speed optimization and caching implementation
  • Mobile responsiveness and cross-browser testing
  • Full analysis of performance bottlenecks based on both real user measurement and lab tests of your production systems.
  • Personalized recommendations of how to change your site architecture to make performance improvements without losing any functionality
  • Recommendations on how to integrate performance reviews into build systems
  • Duration: 2-4 weeks | Investment: $20,000-$40,000

Security Hardening Implementation

  • Review of all public facing systems to catch common security issues
    • URL review to detect possible XSS
    • Public viewable logged error review
    • Content Security Policy, Security headers configuration review
    • Security.txt setup and vulnerability disclosure process
    • Site generator and major Front end library version analysis for latest security patch compliance
  • Identification of missing security audits in public system build pipelines
  • Duration: 1-3 weeks | Investment: $18,000-$28,000
